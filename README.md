# React with Authentication
## Default User
  - Username: `admin`
  - Password: `asd123`

## Start front-end
### don't forget to install node dependencies `npm i`
```
cd frontend
```
```
npm start
```
## Start back-end
### don't forget to install python module `pip install -r requirements.txt`
```
cd backend
```
```
python server.py
```
## Index Page
```
http://localhost:3000
```
## API URL - Swagger UI
```
http://localhost:9999/api/docs
```
